## План доработки портфолио

### 1. Редизайн портфолио

Необходимо обновить общий дизайн портфолио, сделать интерфейс более современным, аккуратным и удобным для пользователя.

Что нужно сделать:

- переработать визуальный стиль сайта;
- улучшить адаптивность под разные устройства;
- заменить текущую секцию `Cases` на новую, более подходящую секцию;
- сделать структуру страницы более понятной и логичной.

---

### 2. Отдельные страницы для проектов

Для каждого проекта необходимо создать отдельную страницу с подробной информацией.

На странице каждого проекта должно быть:

- название проекта;
- большое изображение или mockup проекта;
- подробное описание проекта;
- список использованных технологий;
- ссылка на GitHub-репозиторий;
- ссылка на Live Demo;
- краткое описание функционала проекта.
